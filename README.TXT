# Подземный лабиринт

Текстовый квест на C++ с ASCII-графикой, головоломками, боями и диалогами. Senior версия

## Команда

- Слава И.
- Настя С.
- Слава Д.

## Требования

- Компилятор с поддержкой C++17
- Make (опционально, для удобной сборки)

## Сборка и запуск

### Через Make
Запустить CMD или Powershell в папке со всеми файлами, вписать команду:

make          # сборка
make clean    # очистка
make run      # собрать и запустить

P.S. Отвечать на головоломки на латинице! типо слово - slovo, обещание - obeshaniye.
